Exercise 2: Debugging CSS

Below are the 3 mistakes and what to change. I also used a styled answer container with box-shadow.

Mistake 1: font-family has no fallback fonts.

Fix: font-family: "Times New Roman", Times, serif;

Mistake 2: Missing semicolon after color: red in body.

Fix: color: red;

Mistake 3: Missing semicolon after box-sizing: border-box in .container.

Fix: box-sizing: border-box;

Corrected CSS (clean version):

body {
  font-family: "Times New Roman", Times, serif;
  margin-left: 20%;
  margin-right: 20%;
  text-align: justify;
  background: ivory;
  color: red;
}

h1, h2, h3 {
  color: #ff0000;
  font-size: 2.5em;
  margin: 20px 0px 20px 0px;
}

.container {
  width: 100%;
  padding: 20px;
  box-sizing: border-box;
}

a {
  color: #0000FF;
  text-decoration: underline;
}
